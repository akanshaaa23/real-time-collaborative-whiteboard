# AI-Powered Real-Time Collaborative Whiteboard

A web-based collaborative whiteboard that allows multiple users to draw, write, and interact on a shared canvas in real time. The application includes AI-assisted drawing through voice commands, shape recognition, drag-and-drop editing, and instant synchronization between connected users.

## Features

- Real-time multi-user collaboration
- AI-assisted drawing using voice commands
- Shape recognition and auto-correction
- Text annotations
- Drag-and-drop editing
- Color picker and drawing tools
- Eraser functionality
- Real-time synchronization with Socket.IO
- Responsive user interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Socket.IO
- MongoDB

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project directory

```bash
cd ai-collaborative-whiteboard
```

3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
npm start
```

5. Open the application in your browser.
